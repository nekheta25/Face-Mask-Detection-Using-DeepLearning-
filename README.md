# Face Mask Detection Project 
# 😷 Face Mask Detection Using Deep Learning

## 📌 About the Project
This project is a real-time face mask detection system built using Deep Learning. It detects whether a person is wearing a face mask or not using a webcam.

## 🚀 Features
- Real-time face detection  
- Mask / No Mask classification  
- Fast and efficient performance  

## 🛠️ Technologies Used
- Python  
- TensorFlow / Keras  
- OpenCV  
- NumPy  

## ▶️ How to Run

1. Clone the repository:
git clone https://github.com/nekheta25/Face-Mask-Detection-Using-DeepLearning-.git

2. Go to the project folder:
cd Face-Mask-Detection-Using-DeepLearning-

3. Install dependencies:
pip install -r requirements.txt

4. Run the project:
python mask_detector.py

## 📂 Project Structure
- mask_detector.py → Runs real-time detection  
- mask_train.py → Model training script  
- face_detector/ → Pre-trained face detection model  
- README.md → Project documentation  

## ⚠️ Note
- Dataset is not included due to large file size  
- You can use your own dataset for training  

## 📸 Output
- Green box → Mask  
- Red box → No Mask
